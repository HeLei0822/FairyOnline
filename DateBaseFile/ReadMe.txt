1.	数据库统一用 MySql

2.	先试试能不能直接导入 fairyonline.sql 这个sql 文件（一共26张表，导入的时候少了的话，就是有没成功的地方，再导入几次试试，还是不行的话，来找我）。如果可以的话，应该就是能直接建立数据库了。；如果不行，那就建立一个名字叫 fairyonline 的数据库（不能叫其他名字，不然配置文件找不到），然后导入fairyonline.sql 。（导入SQL文件的方法自行百度）

3.	有些不理解的字段，查看“数据库模块命名1.4”即可，里面对比较生僻的字段，大多是有解析的

4.	具体某个字段的数据类型，打开数据库表看吧（有些记不住了），建立类的时候属性不要建立错了，不然会执行不成功。

5.	建立类的格式根据我写的示例类写就行（没有外键的注解示例，遇到的时候再问我吧），接口命名和方法命名采用驼峰式，各种方法要分类写，Service每种方法后要加入进行简单的注解，一些其他的细节在注解里有，其他的等遇到了再说。
